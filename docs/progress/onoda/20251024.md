### 今日やったこと

- Gitの初期設定・トークン（省略）

### gitコマンド一覧
- コピー：git clone gitのURL
- 現在のbranch：git branch
- ブランチ移動：git branch "ブランチ名"
- ブランチ変更＋移動：git switch -c "ブランチ名"
- コミットしたいものを選択：git add.（git add 相対パス）
- ローカルに保存：git commit -m "コミットメッセージ"
- サーバーにアップロード：git push origin ブランチ名
